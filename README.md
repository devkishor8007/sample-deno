# sample-deno
exploring simple, modern and secure runtime for JavaScript, TypeScript, and Web Assembly i.e. DENO

## Installation

#### Shell (Mac, Linux):
     $curl -fsSL https://deno.land/install.sh | sh

#### PowerShell (Windows):
     $ iwr https://deno.land/install.ps1 -useb | iex
